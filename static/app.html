Setup

First Install

1. Prerequisites

Ensure you have Python 3.10+ installed on your system.

python --version

2. Install Dependencies

Clone the repository and install required packages.

pip install -r requirements.txt

3. Start the Agent

Launch the Aegis local monitoring agent.

python -m agent.cli start --port 8080

4. Configure Your AI App

Set your AI provider's base URL to point at the Aegis proxy and start monitoring all traffic.
