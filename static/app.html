Setup

First Install

1. Prerequisites

Ensure you have Python 3.10+ installed on your system.

python --version

2. Install Dependencies

Clone the repository and install required packages.

pip install -r requirements.txt

3. Start the Agent

Launch the Aegis local monitoring agent.

python -m agent.cli start --port 8080

4. Configure Your AI App

Set your AI provider's base URL to point at the Aegis proxy and start monitoring all traffic.

Cookies Policy

Effective Date: January 1, 2026

Aegis ("we," "our," or "us") uses cookies and similar technologies on our Platform to ensure proper functionality, security, and essential operations. This Cookies Policy explains what cookies we use, why we use them, and your choices regarding them.

1. What Are Cookies

Cookies are small text files stored on your device by your web browser when you visit a website. They enable the website to remember your actions, preferences, and authentication state over a period of time.

2. Essential and Necessary Cookies Only

Aegis uses strictly necessary cookies only. These cookies are required for the Platform to function and cannot be disabled. We do not use any tracking, analytics, advertising, or third-party cookies. The essential cookies we use are:

sessionEssential

Purpose: Maintains your authenticated session across page requests. Without this cookie, you would need to log in on every page load.

Duration: Session (deleted when browser is closed)

csrf_tokenEssential

Purpose: Protects against Cross-Site Request Forgery (CSRF) attacks by ensuring that form submissions originate from our Platform.

Duration: Session (deleted when browser is closed)

aegis_api_keyEssential

Purpose: Stores your API key preference for the Console, allowing you to make authenticated API calls directly from the dashboard without re-entering credentials.

Duration: Session (deleted when browser is closed)

3. No Tracking or Analytics Cookies

Aegis does not use cookies for advertising, analytics, tracking, profiling, or any purpose other than the essential operation of the Platform. We do not share cookie data with third parties. Our logs record request metadata (timestamps, IP addresses, endpoint paths) for security monitoring and audit purposes, but this data is collected server-side and is not stored in cookies.

4. Your Choices

Since we only use essential cookies, disabling them will prevent the Platform from functioning correctly. However, most web browsers allow you to manage cookie settings:

  • Browser settings: You can configure your browser to block or delete cookies. Please note that blocking essential cookies will break core Platform functionality (login, navigation, API key management).
  • Incognito/Private mode: Using private browsing will automatically delete session cookies when you close the browser, which is acceptable for the Platform's operation.

For instructions on managing cookies in your specific browser, visit:

  • Google Chrome
  • Mozilla Firefox
  • Apple Safari
  • Microsoft Edge

5. Changes to This Policy

If we ever need to change the types of cookies we use, we will update this policy and notify you. As a security and governance platform, our commitment is to never use tracking or advertising cookies.

6. Contact

If you have questions about our use of cookies, contact our Data Protection Officer at privacy@aegis.security.
